1. مقدمة سريعة

نظام إلكتروني لتسجيل غياب الطالبات ومتابعته. يعمل من أي جهاز (حاسوب أو هاتف) عبر فتح رابط الموقع في المتصفح.

  • المعلمة: تسجّل غياب طالبات صفوفها يومياً.
  • المديرة: تتابع كل الغيابات، تُخرج التقارير، وتدير الطالبات والمعلمات.
  • مدير النظام: ينشئ حسابات المستخدمين ويحدد صلاحياتهم.

2. تسجيل الدخول (للجميع)

  1. افتحي رابط الموقع في المتصفح (Chrome مثلاً).
  2. أدخلي اسم المستخدم و كلمة المرور.
  3. اضغطي زر تسجيل الدخول.
✅ اسم المستخدم لا يفرّق بين الأحرف الكبيرة والصغيرة (MOH = moh).
✅ كلمة المرور تفرّق بينها — أدخليها تماماً كما هي.
✅ النظام يتذكّر دخولك ويوجّهك للوحة المناسبة لدورك تلقائياً.

4. 👩‍🏫 دليل المعلمة

أ) أول ما ترينه

رسالة ترحيب وتاريخ اليوم، وشبكة بطاقات كل واحدة = صف مسنَد إليك (تعرض اسم الصف، عدد الطالبات، وعدد الغائبات اليوم).

💡 إن ظهرت رسالة «لا توجد صفوف مسندة إليك» — تواصلي مع المديرة لتُسنِد لك صفوفك.

ب) تسجيل غياب طالبة

  1. اضغطي بطاقة الصف (يمكنك البحث عن صف).
  2. بجانب الطالبة الغائبة اضغطي 🔴 تسجيل غياب.
  3. يسألك عن النوع: «موافق» = بعذر · «إلغاء» = بدون عذر.
  4. اكتبي ملاحظة (اختيارية) ثم «موافق».

ج) التراجع عن غياب

اضغطي ↩ تراجع عن الغياب بجانب الطالبة. (تستطيعين التراجع حتى عن غياب سجّلته المديرة لطالبة من صفّك).

د) تفاصيل طالبة وطباعة الكشف

  • ℹ️ تفاصيل: لرؤية بيانات الطالبة (عرض فقط).
  • 🖨️ طباعة كشف غيابي: كشف رسمي بكل غيابات اليوم.
  • ↩ خروج: للخروج.

🚫 ما لا تستطيع المعلمة فعله

  • إضافة/تعديل/حذف طالبات.
  • رؤية أو إدارة معلمات أخريات.
  • تسجيل غياب لطالبة خارج صفوفها المسندة.
  • رؤية التقارير العامة أو الإعدادات.

5. 👩‍💼 دليل المديرة

الشريط العلوي: 📋 سجل الغيابات 🧑‍🎓 تقرير الطالبات 📊 التقارير ⚙️ إدارة النظام 🔒 كلمة المرور ↩ خروج

أ) 📋 سجل الغيابات

  • الفلاتر: المعلمة · الصف · الشعبة · التخصص · التاريخ · بحث بالاسم (وزر «إعادة تعيين»).
  • تبديل النوع: ↑ بعذر / ↓ بدون عذر — مثلاً عند وصول تقرير طبي.
  • حذف · 🖨️ طباعة كشف · 📥 تصدير CSV.

ب) 🧑‍🎓 تقرير الطالبات

  • اختيار طالبات محددات: اكتبي جزءاً من اسم أي طالبة واختاريها (واحدة أو أكثر). تركها فارغة = كل من لها غياب.
  • أزرار الفصول + فلاتر التاريخ/الصف/الشعبة/التخصص + خيار «المتجاوزات فقط».
  • الأعمدة: إجمالي الغياب · بعذر · بدون عذر · نسبة الحضور · الحالة (🚨 تجاوزت السقف بالأحمر).
  • أزرار 🖨️ طباعة و 📥 تصدير CSV.

ج) 📊 التقارير الإحصائية

  • فترات سريعة (اليوم/الأسبوع/الشهر/الكل)، مؤشرات، رسم بياني للاتجاه.
  • جدول أكثر الطالبات غياباً + تجميعات حسب الصف/الشعبة/التخصص/المعلمة.
  • أزرار الطباعة والتصدير.

د) ⚙️ إدارة النظام (4 تبويبات)

1) الطلاب

  • إضافة: املئي الحقول ثم ➕ إضافة طالب. الصف والشعبة قوائم منسدلة (ولإضافة جديدة اختاري «➕ إضافة جديد…»).
  • نقل لشعبة / تعديل: ابحثي عنها ← تعديل ← غيّري الشعبة ← 💾 حفظ التعديل.
  • حذف · تفاصيل · استيراد من Excel.
⚠️ استيراد Excel يستبدل قائمة الطالبات الحالية بالكامل — استخدميه بحذر.

2) المعلمات (إسناد الصفوف)

  • أضيفي معلمة (اسم + اسم مستخدم + كلمة مرور).
  • إسناد الصفوف: اختاري صفوف المعلمة (مع بحث وأزرار «تحديد الكل» / «إلغاء الكل»). بدون إسناد لن ترى المعلمة أي طالبة.

3) 🔑 كلمات السر

قائمة بكل المستخدمين، وبجانب كل واحد زر 🔑 إعادة تعيين لوضع كلمة سر جديدة دون الحاجة للقديمة (مفيد عند نسيان المعلمة كلمتها).

4) ⚙️ الإعدادات

  • سقف الغياب بدون عذر: العدد الذي تجاوزه يُظهر مؤشّر تحذير (مثلاً 20).
  • الفصول الدراسية: اسم وتاريخ بداية/نهاية كل فصل ثم 💾 حفظ الإعدادات.

⭐ خطوة أولى مهمة جداً: إسناد الصفوف للمعلمات

⚠️ قبل أن يعمل النظام للمعلمات، يجب على المديرة إسناد الصفوف لكل معلمة. المعلمة بلا صفوف مسندة ترى رسالة «لا توجد صفوف مسندة إليك» ولا تستطيع تسجيل أي غياب.

كيف تُسنِد المديرة الصفوف لمعلمة (خطوة بخطوة):

  1. ادخلي كمديرة، ثم اضغطي ⚙️ إدارة النظام.
  2. اضغطي تبويب المعلمات.
  3. في قائمة المعلمات بالأسفل، استخدمي خانة 🔍 البحث لإيجاد المعلمة، ثم اضغطي تعديل بجانبها.
  4. في قسم «الصفوف/الشعب المسندة»:
    • استخدمي خانة 🔍 ابحث عن صف أو شعبة للوصول السريع.
    • اضغطي على الصفوف التي تُدرّسها المعلمة (تظهر بعلامة ✓).
    • أو استخدمي تحديد الكل إن كانت تُدرّس كل الصفوف الظاهرة.
  5. اضغطي 💾 حفظ التعديل.
💡 نصيحة لتسريع العمل مع 66 معلمة: جهّزي ورقة فيها اسم كل معلمة والصفوف التي تُدرّسها، ثم أسنديها معلمة تلو الأخرى. كل معلمة تستغرق أقل من دقيقة.
✅ بمجرد إسناد الصفوف لمعلمة، تستطيع الدخول فوراً ورؤية طالباتها وتسجيل غيابهنّ.

⭐ ملاحظة أمنية: أول دخول لكل مستخدم

عند أول دخول لأي مستخدم (مديرة أو معلمة)، سيطلب منه النظام تغيير كلمة المرور قبل المتابعة — لا يمكنه استخدام النظام حتى يختار كلمة مرور خاصة به.

  • كلمة المرور المؤقتة لكل المستخدمين حالياً: 1234
  • عند الدخول، تظهر نافذة «🔒 يجب تغيير كلمة المرور» — أدخلي الحالية (1234) والجديدة مرتين.
  • هذا يحمي بيانات الطالبات ويضمن أن لكل مستخدم كلمة سر خاصة وسرّية.
📌 إذا نسيت معلمة كلمة مرورها لاحقاً: المديرة ← ⚙️ إدارة النظام ← 🔑 كلمات السر ← «إعادة تعيين». (سيُطلب من المعلمة تغييرها مجدداً عند الدخول التالي).

💾 النسخ الاحتياطي (للمديرة)

للحفاظ على بياناتكم، نزّلي نسخة احتياطية بشكل دوري (أسبوعياً مثلاً):

  1. ⚙️ إدارة النظام ← تبويب ⚙️ الإعدادات.
  2. في قسم «النسخ الاحتياطي»، اضغطي ⬇️ تنزيل نسخة احتياطية.
  3. احفظي الملف في مكان آمن (حاسوب المدرسة / Google Drive).

للاستعادة عند الحاجة: ⬆️ استعادة من ملف واختاري ملف النسخة.

⚠️ الاستعادة تستبدل كل البيانات الحالية — استخدميها فقط عند الضرورة.

6. 🛠️ دليل مدير النظام

مسؤول عن حسابات المستخدمين (يرى إحصاءاتهم).

إضافة مستخدم

  1. املأ: الاسم · اسم المستخدم (إنجليزي) · كلمة المرور.
  2. اختر الدور: مديرة أو معلمة.
  3. اضغط ➕ إضافة مستخدم.

تعديل / حذف

  • تعديل: لتغيير الاسم/الدور/كلمة المرور (اتركها فارغة للإبقاء عليها).
  • حذف: لحذف المستخدم (مع تأكيد).

7. 🔒 تغيير كلمة المرور (للجميع)

  1. اضغطي 🔒 كلمة المرور أعلى الصفحة.
  2. أدخلي: الحالية · الجديدة · تأكيد الجديدة.
  3. اضغطي تغيير.
الشروط: الجديدة 4 أحرف على الأقل ويجب أن تطابق التأكيد.

8. ❓ أسئلة شائعة

المعلمة دخلت لكن لا ترى أي صف؟

لم تُسنَد لها صفوف. المديرة ← ⚙️ إدارة النظام ← المعلمات ← تعديل ← اختيار الصفوف ← حفظ.

المعلمة نسيت كلمة مرورها؟

المديرة ← ⚙️ إدارة النظام ← 🔑 كلمات السر ← «إعادة تعيين» بجانب المعلمة.

كيف أنقل طالبة لشعبة أخرى؟

المديرة ← ⚙️ إدارة النظام ← الطلاب ← ابحثي عنها ← تعديل ← غيّري الشعبة ← حفظ التعديل.

ما الفرق بين «بعذر» و «بدون عذر»؟

«بعذر» (تقرير طبي) لا يُحتسب ضمن سقف الإنذار. «بدون عذر» يُحتسب، وعند تجاوز السقف يظهر مؤشر 🚨.

وصل تقرير طبي بعد تسجيل الغياب بدون عذر؟

المديرة ← سجل الغيابات ← بجانب الغياب اضغطي «↑ بعذر» لتحويله.

هل يعمل النظام على الهاتف؟

نعم، النظام مصمّم للهاتف والحاسوب — افتحي الرابط في المتصفح.
